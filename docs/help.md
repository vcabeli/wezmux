## Getting Help

This is a spare time project, so please bear with me. There are a couple of channels for support:

* Use the GitHub issue tracker to see if someone else has a similar issue, or to file a new one: <https://github.com/vcabeli/wezmux/issues>
* Start or join a thread in [GitHub Discussions](https://github.com/vcabeli/wezmux/discussions) if you have general questions or want to talk through usage or roadmap.

GitHub Discussions are better suited to questions than bug reports, but don't
be afraid to use whichever channel fits best and we'll work it out.
