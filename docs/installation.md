Wezmux currently documents and packages a macOS-first workflow, with Windows
available as a preview zip artifact.

The repository still contains the broader upstream WezTerm source and packaging
layout, but Linux and BSD packaging paths should be treated as inherited
internals until Wezmux-specific support is documented and released.

 - [Windows preview](install/windows.md)
