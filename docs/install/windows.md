## Installing on Windows

Windows support is currently published as a preview zip artifact.

1. Download `Wezmux-windows-x64.zip` from the latest release or from the
   `windows` workflow artifact.
2. Extract the zip.
3. Run `wezterm-gui.exe`.

The preview zip includes the runtime DLLs needed by the inherited WezTerm
Windows backend. It does not install shell integrations, PATH entries, context
menus, winget metadata, or Wezmux agent hook wrappers yet.
